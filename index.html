CSS syntax ประกอบด้วยอะไรบ้าง
  • Selector
  • Property
  • Value
การเขียน CSS มีกี่แบบอะไรบ้าง
  • Inline
  • Exter
  • Inter
selector มีกี่แบบอะไรบ้าง
  • Class
  • Id
  • Pseudo Class
  • Pseudo Element
combinator มีกี่แบบอะไรบ้าง
  • Descendant combinator
    • example: body h1
  • Combinator
    • example: p>button
  • Adjacent sibling combinator
    • example: p+img
  • General sibling combinator
    • example: p ~ img
property และ value คืออะไรแตกต่างกันอย่างไร
  • value คือค้่าของ property
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
  • Content eg. font-size,font-weigh,font-family,color
  • Border eg. border table
  • Padding eg. padding-left padding-right padding-top padding-bottom
  • Margin margin-left margin-right margin-top margin-bottom
  • Element eg. height,weight,flex,grid
value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร
  • Absolute eg px
  • Relative eg rem em
